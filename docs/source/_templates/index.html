{# ProbNum's features #}

Solve Numerical Problems

Solve problems from linear algebra, optimization, quadrature and differential equations using probabilistic inference.

Quantify Uncertainty in Computation

Quantify and propagate uncertainty arising from finite resources and stochastic input through computational pipelines.

Compose Custom Numerical Methods

Create a problem-specific probabilistic numerical method from a pre-defined set of components or implement your own.

{# Installation #}

Installation

To get started install ProbNum using pip. 
pip install probnum

Then in a Python session simply import ProbNum.

import probnum as pn
